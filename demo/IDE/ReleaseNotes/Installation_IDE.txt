Installation d'IDE:

1er Cas: IDE depuis Sourceforge.net
Il existe une foire aux questions relative à où et comment installer IDE. Si vous n'êtes pas un
nouvel utilisateur, ces informations ne seront sans doute pas utiles.

IDE peut être installé dans un répertoire qui peut être relié à celui d'Euphoria, ou dans quelque chose
de complètement différent si vous établi des associations de fichiers Windows. Le nom de répertoire 
importe peu. Je ne recommande pas d'installer IDE dans l'un des répertoires créés par Euphoria lors 
de son installation, ni dans des sous-repertoires de ceux-ci. Toutefois, quelque chose du genre
Euphoria\IDE devrait fonctionner sans problème. Certains utilisateurs ont mis IDE dans 
Program Files ou son équivalent.

Décompresser IDE dans le répertoire choisi. Ne changez pas les sous répertoires qui seront créés, 
sans quoi IDE ne fonctionnera pas correctement.

Cette version nécessite Win32lib v0.70.4a ou plus.

Vous pourrez avoir besoin de trois bibliothèques supplémentaires, que vous trouverez sur la 
page Recent Contributions d'Euphoria. Il s'agit de systray.ew, alphablend.ew et EuGrid.ew.
Elles ne servent que si vous voulez, respectivement, minimiser votre fenêtre dans l'aire de 
notification plutôt que la barre de tâches, disposer de fenêtres translucides ou utiliser le 
composant Eugrid.

Si votre répertoire de win32lib n'est pas Euphoria\Include, et n'est pas listé dans la variable 
d'environnement EUINC, vous devrez placer ou copier le contenu du sous-répertoire "includes"
de win32lib dans le répertoire où vous avez installé IDE.

Lorsque vous faites une mise à jour de win32lib, assurez-vous d'utiliser la dernière version
d'IDE, car les fchiers peuvent avoir changé.

2e Cas: IDE lié distribué par Andy Drummond: il contient tous les fichiers nécessaires pour
utiliser IDE.exe.
