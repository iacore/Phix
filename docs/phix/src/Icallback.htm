Icallback

Syntax: include pIUP.e

Icallback("routine")
Description: Utility function. Shorthand for call_back(routine_id(s)).
Comments: The routine name must be in scope and already defined at the point of use.
Example: 
include pIUP.e

function quit_cb()
  return IUP_CLOSE;
end function

Ihandle quit_bt = IupButton("Quit", 0)
IupSetCallback(quit_bt, "ACTION", Icallback("quit_cb"))
See Also: Ihandle, IupButton, IupSetCallback
