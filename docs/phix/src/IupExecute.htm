IupExecute

Definition: include pGUI.e

integer res = IupExecute(string filename, nullable_string parameters=NULL)
-- or --
integer res = IupExecuteWait(string filename, nullable_string parameters=NULL)
Description: Runs the executable with the given parameters.
Included for compatibility only, the recommendation is to use system_exec() instead, or perhaps one of its brethren.

filename: name of the executable. Can contain a path.
parameters: optional parameters.

IupExecute is a non synchronous operation, i.e. the function returns just after executing the command without waiting for a result, whereas IupExecuteWait is a synchronous operation, i.e. the function waits for the command to terminate before it returns.

In Windows, there is no need to add the ".exe" file extension.

Returns: 1 if successful, -1 if failed. In Windows and GTK can return -2 if file not found.

IupExecute() is used by the IupHelp() function.
See Also: system_exec
