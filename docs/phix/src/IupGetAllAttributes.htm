IupGetAllAttributes

Syntax: include pGUI.e

sequence names = IupGetAllAttributes(Ihandle ih)
Description: Returns a list of the names of all attributes of an element that are set in its internal hash table only.
Notes: The C function IupGetAttributes should be avoided, since I believe it can fail if any non-string attributes exist, and this routine used instead.
See Also: IupGetAttribute, IupSetAttribute, IupSetAttributes
