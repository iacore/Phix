IupGetBrother

Definition: include pGUI.e

Ihandln res = IupGetBrother(Ihandle ih, bool bPrev=false)
Description: Returns either brother of an element.

ih: identifier of the interface element.
bPrev: return the next (default) or previous brother.

Note that bPrev is a Phix/pGUI.e (and pGUI.js) addition to the C API. Specifying a bPrev of true is useful in that it can, for instance, return a preceding label before an input field.

Returns: the brother or NULL if there is none.
pwa/p2js: Supported.
See Also: IupGetChild, IupGetNextChild, IupGetParent
