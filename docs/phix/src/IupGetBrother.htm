IupGetBrother

Syntax: include pIUP.e

Ihandln res = IupGetBrother(Ihandle ih)
Description: Returns the brother of an element.

ih: identifier of the interface element.

Returns: the brother or NULL if there is none.
See Also: IupGetChild, XIupOpen, IupGetParent
See Also IupGetChild, IupGetNextChild, IupGetParent
