IupGetFile

Syntax: include pGUI.e

nullable_string result = IupGetFile(string filefilter)
Description: Shows a modal dialog of the native interface system to select a filename. Uses the IupFileDlg element.

filefilter: This parameter is used as an input value to define the default filter and directory, eg "../docs/*.txt".

Returns: a string contrainig the selected file or NULL.
Notes: The result string is limited to 4096 characters.

The function will reuse the directory from one call to another, so in the next call will open in the directory of the last selected file.

The dialog uses a global attribute called "PARENTDIALOG" as the parent dialog if it is defined.
It also uses a global attribute called "ICON" as the dialog icon if it is defined.

For a more controlled dialog use directly the IupFileDlg element.
See Also: IupFileDlg, IupMessage, xIupScanf, IupListDialog, IupAlarm, xIupSetLanguage
See Also IupFileDlg, IupMessage, *IupScanf, IupListDialog, IupAlarm, *IupSetLanguage.
