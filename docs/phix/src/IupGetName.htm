IupGetName

Syntax: include pGUI.e

nullable_string name = IupGetName(atom ih)
Description: Returns the name of an interface element, if the element has an associated name using IupSetHandle()

ih: Identifier of the interface element.

Returns: the name, or NULL if no name has been associated with the element.
See Also: IupSetHandle, IupGetHandle, IupGetAllNames
Technicalia If more than one name has been associated with the specified element, it returns the first one found, in a non-deterministic fashion with no guarantee of any ordering (such as first or last, chronological or alphabetical: you may as well assume it will be random and different on each run). As such it is not something that could ever be improved, but something for you to not do.
