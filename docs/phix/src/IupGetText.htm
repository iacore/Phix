IupGetText

Syntax: include pGUI.e

nullable_string result = IupGetText(string title, string text)
Description: Shows a modal dialog to edit a multiline text.

text: the initial value of the text.

Returns: the modified text or NULL if cancelled.
Notes: The string is limited to 10240 characters (about 250..300 lines of average length).

The dialog uses a global attribute called "PARENTDIALOG" as the parent dialog if it is defined.
It also uses a global attribute called "ICON" as the dialog icon if it is defined.
