IupLoadImage

Syntax: include pIUP.e

atom ih = IupLoadImage(string filename)
Description: filename: Name of the file to be loaded.

Returns: the identifier of the created image, or NULL if an error occurs. When it fails the global attribute "IUPIM_LASTERROR" is set with a message describing the error. When it succeeds a valid Ihandle is returned, but in pIUP storing NULL in a variable declared as Ihandle triggers a typecheck.
