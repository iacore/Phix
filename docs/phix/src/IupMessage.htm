IupMessage

Syntax: include pGUI.e

IupMessage(nullable_string title = NULL, nullable_string msg = NULL)
Description: Shows a modal dialog containing a message with an OK button.
Comments: If title is null it defaults to "Error". If msg is null then a window with just the title and an OK button is displayed.

Uses global attributes "PARENTDIALOG" and "ICON" (but not on Windows as MessageBox does not have an icon in the title bar) if defined.

In the binaries currently shipped, IupMessage can fail if called without a main window. This has been fixed in the latest source code (and not deemed important enough for me to download the sources, a C compiler, blah blah blah, and mess something else up).
Example: 
include pGUI.e

IupOpen()

IupMessage("Oops","It went wrong.")

IupClose()
