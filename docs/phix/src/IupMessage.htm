IupMessage

Syntax: include pIUP.e
IupMessage(object title = NULL, object msg = NULL)
Description: Shows a modal dialog containing a message with an OK button.
Comments: If title is not defined (/left null) it defaults to "Error". If msg is not defined (/left null) then a window with just the title and an OK button is displayed.

Uses global attributes "PARENTDIALOG" and (but not on Windows as MessageBox does not have an icon in the title bar) "ICON" if defined.

In the binaries currently shipped, IupMessage can fail if called without a main window. This has been fixed in the latest source code (and not deemed important enough for me to download the sources, a C compiler, blah blah blah, and mess something else up).
Example: 
include pIUP.e

IupOpen()

IupMessage("Oops","It went wrong.")

IupClose()
