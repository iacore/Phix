IupMessageError

Definition: include pGUI.e

IupMessageError(Ihandln parent, string message, bool bWrap=true)
Description: Shows a modal dialog containing an error message. It simply creates and pops up an IupMessageDlg with DIALOGTYPE=ERROR.

parent: parent dialog, can be NULL.
message: text message contents. It can be a language pre-defined string without the "_@" prefix. bWrap: convert each paragraph into a single line, thereby delegating wordwrap to the function.
pwa/p2js: Not supported. See also the comments on IupMessage.
See Also: IupGetFile, IupGetParam, IupListDialog, IupAlarm, IupMessage, IupMessageDlg
Notes: If parent is NULL the title defaults to "Error!" and tries the global attribute "PARENTDIALOG" as the parent dialog.

The dialog title will be the same title of the parent dialog.

The dialog is shown centered relative to its parent.
