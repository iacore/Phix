IupRedraw

Syntax: include pGUI.e

IupRedraw(Ihandles ih, bool children)
Description: Force the element and optionally its children to be redrawn immediately.

ih: identifier of the interface element, or a sequence of elements. children: flag to update its children. Can be 0 or 1.
Notes: If ih is a sequence, the one single value of children is used for each of the elements it contains.
See Also: IupUpdate
