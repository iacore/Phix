IupSetAttributes

Syntax: include pIUP.e

IupSetAttributes(Ihandle ih, string attributes, sequence data = {})
Description: Sets several attributes of an interface element.

ih: Identifier of the interface element.
attributes: string with the attributes in the format "v1=a1, v2=a2,..." where vi is the name of an attribute and ai is its value.
data: if a non-empty sequence is passed, it performs attributes = sprintf(attributes,data) first.
Comments: The function IupSetAttributesf is identical, except that it returns the ih parameter, which can make it easier to create dialogs.

Of course there are cases such as IupSetStrAttribute(ih,"TITLE","this,that [SIZE=%d]",data) which cannot be achieved via IupSetAttributes, because of the obvious mis-parsing of commas and equal signs, but also bear such in mind, especially when data originates from file or user input, as a similar potential catastrophy might not be so readily apparent.
Example 1: 
canvas = IupSetAttributesf(IupCanvas(NULL), 
                           "BORDER=NO, RASTERSIZE=100x100")
dialog = IupSetAttributesf(IupDialog(IupSetAttributesf(IupHBox({canvas}),
                                                       "MARGIN=10x10")),
                           "TITLE=Test")
Example 2: Creates a list with country names and defines Japan as the selected option. 
Ihandle list = IupSetAttributesf(IupList(NULL),
                                 "VALUE=3,"&
                                 "1=Brazil,"&
                                 "2=USA,"&
                                 "3=Japan,"&
                                 "4=France");
