IupSetCallback

Syntax: include pIUP.e

IupSetCallback(Ihandle ih, string name, atom func)
or ...
Ihandle ih = IupSetCallbackf(Ihandle ih, string name, atom func)
Description: Associates a callback to an event.

ih: identifier of the interface element.
name: name of the callback.
func: a callback address, usually specified as Icallback("name"). If NULL removes the association.
Notes: This function replaces the deprecated combination: 
IupSetFunction(global_name, func)
IupSetAttribute(ih, name, global_name)

So it eliminates the need for a global name.

Callbacks set using IupSetCallback can not be retrieved using IupGetFunction.

The C function returns the address of the previous function associated to the action, whereas IupSetCallbackf returns ih, which can make it easier to declare dialogs.
See Also: XIupOpen, XIupClose
