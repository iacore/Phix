ZipClose

Definition: include builtins\LiteZip.e

ZipClose(object hzip)
Description: Closes the zip archive(s). The parameter hzip may be an atom or a [flat] sequence of atoms.

Terminates in error on failure.
Example:
Copy 
include builtins\LiteZip.e

atom hzip = ZipCreateFile("test.zip")

ZipClose(hzip)

--or--

sequence hzips = {ZipCreateFile("part1.zip"),
                  ZipCreateFile("part2.zip")}
-- quick check for any errors:
if sum(sq_atom(hzips))!=length(hzips) then ?9/0 end if

ZipClose(hzips)
See Also: ZipCreateFile
