abs

Syntax: object x2 = abs(object x1)
Description: Calculate the absolute (unsigned) value of x1.
Comments: This function may be applied to an atom or all elements of a sequence.

Note: Unlike several other functions there is no separate sq_abs function, instead the one routine (see builtins\pabs.e) can be used for any argument.
Example: 
r = abs(-4)
-- r is 4
