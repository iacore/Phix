allocate_string

Syntax: atom a = allocate_string(string st)
Description: Allocate space for string sequence st. Copy st into this space along with a 0 terminating character. This is the format expected for C strings. The memory address of the string will be returned. If there is not enough memory available, 0 will be returned.
Comments: To free the string, use free().

Phix strings can be passed directly to C functions, however doing so is not compatible with OpenEuphoria.
Example: 
atom title
title = allocate_string("The Wizard of Oz")
...
free(title)
Example Program: demo\arwendemo\mmsd.exw
See Also: allocate, free
