cdCanvas

Syntax: include pGUI.e

cdCanvas identifier
Description: Type. An integer > 0.

Use this to define CD canvas variables. It is probably unwise to use this to determine whether an object is an cdCanvas, as many things which are clearly not (eg 1, 5000, 'X') will pass.
Comments: The type imImage is defined in a near-identical fashion, but obviously used for different purposes.
Example: 
include pGUI.e

cdCanvas hCdCanvas = cdCreateCanvas(CD_IUP, IupCanvas())
See Also: IupCanvas, cdCreateCanvas
