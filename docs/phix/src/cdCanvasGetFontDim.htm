cdCanvasGetFontDim

Definition: include pGUI.e

integer {width, height, ascent, descent} = cdCanvasGetFontDim(cdCanvas hCdCanvas)
-- or --
atom {width, height, ascent, descent} = wdCanvasGetFontDim(cdCanvas hCdCanvas)
Description: Returns the maximum width of a character, the line height, the ascent and descent of the characters of the currently selected font.

The line height is the sum of the ascent and descent plus a given additional space (if this is the case).
All values are given in pixels and are positive.
