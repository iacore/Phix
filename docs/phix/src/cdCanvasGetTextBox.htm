cdCanvasGetTextBox

Syntax: include pGUI.e

atom {xmin, xmax, ymin, ymax} = cdCanvasGetTextBox(cdCanvas hCdCanvas, atom x, atom y, string text)
-- or --
atom {xmin, xmax, ymin, ymax} = wdCanvasGetTextBox(cdCanvas hCdCanvas, atom x, atom y, string text)
Description: Returns the horizontal bounding rectangle occupied by a text at a given position.
Comments: If orientation is not 0 then its area is always larger than the area of the rectangle returned by cdCanvasGetTextBounds.

Since the pGUI wrapper functions can accept integers and floats in the parameters, and return atoms or integers, there is no need for a separate cdfCanvasGetTextBox wrapper. Internally cdCanvasGetTextBox uses the cdfCanvasGetTextBox C function.
