cdCanvasText

Definition: include pGUI.e

cdCanvasText(cdCanvas canvas, atom x, y, string text)
Description: Draws a text in the position (x,y) according to the current font and text alignment. It expects an ANSI string. Can have line breaks.

The cdText() routine ceased to be available in IUP version 3.25.
Calls to cdfCanvasText should simply have the 'f' deleted.
wdCanvasText() is identical, except that x and y are in world coordinates.
