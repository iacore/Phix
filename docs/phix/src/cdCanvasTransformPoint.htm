cdCanvasTransformPoint

Syntax: include pGUI.e

atom {tx,ty} = cdCanvasTransformPoint(cdCanvas hCdCanvas, atom x, atom y)
Description: Applies a transformation to a given point.
Comments: Since the pGUI wrapper function can accept integers and floats in the parameters, and return atoms or integers, there is no need for a separate cdfCanvasTransformPoint wrapper. Internally it only uses the cdfCanvasTransformPoint C function.
