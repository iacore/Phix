cdEncodeColor

Definition: include pGUI.e

integer color = cdEncodeColor(integer red, integer green, integer blue)
Description: Returns a codified triple (r,g,b) in a long integer such as 0x00RRGGBB, where RR are the red components, GG are the green ones and BB are the blue ones.
The code is used in the CD library to define colors. It can be used without an active canvas.
