ceil

Syntax: atom x2 = ceil(atom x1)
Description: round x1 up to the next integer.
Comments: For integers, ceil() returns the same result as floor().

For non-integers, ceil() is one more than floor().

This function may be applied to an atom or sq_ceil() to all elements of a sequence. The rules for sequence operations apply.
Example: 
r = ceil(1.4)           -- r is 2
r = ceil(-1.4)          -- r is -1
See Also: floor
