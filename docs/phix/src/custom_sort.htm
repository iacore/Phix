custom_sort

Syntax: sequence s2 = custom_sort(integer rid, sequence s1)
Description: Sort the elements of sequence s1, using a compare function with routine id rid.
Comments: Your compare function must be a function of two arguments similar to the builtin compare(). It must compare two objects and return -1, 0 or +1.
Example Program: demo\csort.ex
See Also: sort, compare, routine_id, tagset
