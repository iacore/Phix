format_timedate

Syntax: string s = format_timedate(timedate td, string fmt=default_format)
Description: Convert a timedate into a string.
Comments: The td parameter is normally a result originally from parse_date_string(), posibly adjusted. You can also legally pass the result of date() to this routine, as long as fmt does not contain a timezone. The fmt parameter is a single format strings, whose default value can be set via set_formats().

It is perfectly acceptable to create and post-process several fragments before stitching them all together for the final result. The only exception is that an am/pm indicator must be in the same fragmnent as the hour.

You may also, performance considerations aside, feel that 
                        if format_timedate(td,"Dddd")="Monday" then

is clearer than the equivalent 
                        if td[DT_DOW]=1 then

Example: 
?format_timedate(td,"'Today''''s date is' Dddd, Mmmm ddth, YYYY")
-- output is eg eg "Today's date is Monday, September 3rd, 2012".
See Also: timedate, format strings, set_formats
