gGetBrother

Definition: include xpGUI.e

gdx res = gGetBrother(gdx id, bool bPrev=false)
Description: Returns either brother of an element.

id: identifier of the interface element.
bPrev: return the next (default) or previous brother.

Specifying a bPrev of true is useful in that it can, for instance, return a preceding label before an input field.

Returns: the brother or NULL if there is none.
pwa/p2js: Supported.
See Also: gGetChild, gGetParent
