gGetDialog

Definition: include xpGUI.e

gdx res = gGetDialog(gdx id)
Description: Returns the handle of the dialog that contains that interface element. (not yet tested:) Works also for children of a menu that is associated with a dialog.

id: Identifier of an interface element.

Returns: an gdx, the identifier of the dialog or NULL if not found.
pwa/p2js: Supported. Not that it should matter, but the result is an HtmlElement rather than an integer id.
