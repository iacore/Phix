get_rand

Definition: atom seed = get_rand()
Description: Obtain the current random number generator seed, which can be passed back to set_rand() to replicate things.
Comments: Sometimes, a program crashes every 10th or 20th time it is run.
A suitably placed "atom save = get_rand()" can make it much easier to replicate and debug the problem by retrieving such a saved seed from the ex.err file.

Fairly obviously, set_rand(get_rand()) is not going to achieve anything, then again it is not going to hurt anyone either.
pwa/p2js: Supported.
Example: 
sequence {s, t} @= repeat(0, 3)
atom save = get_rand()
s[1] = rand(10)
s[2] = rand(100)
s[3] = rand(1000)
set_rand(save)  -- restore internal seed
t[1] = rand(10)  -- same arguments to rand() as before
t[2] = rand(100)
t[3] = rand(1000)
-- at this point s and t will be identical
