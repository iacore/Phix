get_screen_char

Syntax: include image.e
sequence s = get_screen_char(integer line, integer column)
Description: Return a 2-element sequence s, of the form {ascii-code, attributes} for the character on the screen at line, column. s consists of two atoms. The first is the ASCII code for the character. The second is an atom that contains the foreground and background color of the character, and possibly other information describing the appearance of the character on the screen.
Comments: With get_screen_char() and put_screen_char() you can save and restore a character on the screen along with its attributes.
Example: 
-- read character and attributes at top left corner
s = get_screen_char(1,1) 
-- store character and attributes at line 25, column 10
put_screen_char(25, 10, {s})
See Also: put_screen_char, save_text_image
