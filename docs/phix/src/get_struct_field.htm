get_struct_field

Syntax: atom v = get_struct_field(integer id, atom pStruct, string fieldname)
Description: get a named field from a structure previously allocated by allocate_struct().
Comments: The return value is always an atom. You may need to use say peek_string() on some results.
Example: 
include cffi.e
atom pTitle = get_struct_field(idMBP,pMBP,"lpszText")
?peek_string(pTitle)
See Also: allocate_struct, set_struct_field, peek_string
