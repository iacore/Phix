getd_all_keys

Definition: sequence res = getd_all_keys(integer tid=1)
Description: Retrieve all keys present in the specified dictionary.

tid can be omitted, to use the default dictionary, or the result of a previous new_dict() call.
Comments: Ideal when you don’t really need the data, just the keys (in order), and obviously simpler than using traverse_dict().
Example: 
setd("this",11)
setd("that",7)
setd("those",33)
?getd_all_keys()    -- prints {"that","this","those"}
See Also: getd, traverse_dict
