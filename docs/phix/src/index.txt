--  an error occurs if url does not contain ".htm"
-- blank lines and any beginning with any of -=>! are ignored,
-- which can freely be used to organise this into sections.
--  (I use -- for Ctrl Q sections and > for F4 jumping, and
--   ! to omit things that I ought to get back to later.)
-- you can also indent things (ignored), if that helps any.
