int_to_bytes

Syntax: sequence s = int_to_bytes(atom a)
Description: Convert an integer into a sequence of 4 bytes. These bytes are in the order expected on the 386+, i.e. least-significant byte first.
Comments: You might use this routine prior to poking the 4 bytes into memory for use by a machine language program.

The integer can be negative. Negative byte-values will be returned, but after poking them into memory you will have the correct (twos complement) representation for the 386+.

This function will correctly convert integer values up to 32-bits. For larger values, only the low-order 32-bits are converted. The integer type only allows values up to 31-bits, so declare your variables as atom if you need a larger range.
Example 1: 
s = int_to_bytes(999)
-- s is {231, 3, 0, 0}
Example 2: 
s = int_to_bytes(-999)
-- s is {-231, -4, -1, -1}
See Also: bytes_to_int, int_to_bits, bits_to_int, peek, poke
