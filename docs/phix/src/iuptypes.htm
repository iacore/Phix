Types

pGUI uses a relatively small set of types plus one pertinent helper utility:

Icallback - utility to declare a callback, and enable validation of the same.
Ihandle - an interface element, such as a button, image, timer, container, or whole dialog.
an Ihandln is an Ihandle or NULL. Use suggests test/error handling should be present.
cdCanvas - a drawing canvas, on the screen/clipboard/printer/file/etc.
nullable_string - a string or NULL. Use suggests test/error handling should be present.
