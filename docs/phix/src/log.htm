log

Syntax: atom x2 = log(atom x1)
Description: Return the natural logarithm of x1.
Comments: This function may be applied to an atom or sq_log() to all elements of a sequence. The rules for sequence operations apply.

Note that log is only defined for positive numbers. Your program will abort with a message if you try to take the log of a negative number or zero.
Example: 
a = log(100)
-- a is 4.60517
See Also: sin, cos, tan, sqrt
