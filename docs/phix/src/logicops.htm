Logical Operators

The logical operators and , or , xor , and not are used to determine the "truth" of an expression. e.g. 
        1 and 1     -- 1 (true)
        1 and 0     -- 0 (false)
        0 and 1     -- 0 (false)
        0 and 0     -- 0 (false)
        1 or 1      -- 1 (true)
        1 or 0      -- 1 (true)
        0 or 1      -- 1 (true)
        0 or 0      -- 0 (false)
        1 xor 1     -- 0 (false)
        1 xor 0     -- 1 (true)
        0 xor 1     -- 1 (true)
        0 xor 0     -- 0 (false)
        not 1       -- 0 (false)
        not 0       -- 1 (true)
You can also apply these operators to numbers other than 1 or 0. The rule is: zero means false and non-zero means true. So for instance: 
        5 and -4     -- 1 (true)
        not 6       -- 0 (false)
Function-style equivalents of these operators can also be applied to sequences. See below.

In all cases short-circuit evaluation is used for expressions containing and or or .

See also and_bits, or_bits, and xor_bits, which are the corresponding bitwise functions.
