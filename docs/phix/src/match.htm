match

Syntax: integer i = match(sequence s1, sequence s2, integer start=1)
Description: Try to match s1 against some slice of s2. If successful, return the element number of s2 where the (first) matching slice begins, else return 0.

The match_from() routine, which predated optional parameters, is now deprecated. Simply use match() with the optional third parameter instead.
Example 1: 
location = match("hi", "Phix") -- location is set to 2
Example 2: 
location = match("the", "the cat sat on the mat", 4)
-- location is set to 16, not 1, because a start index of 4 was specified
See Also: find, compare, wildcard_match
