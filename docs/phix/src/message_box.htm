message_box

Platform: WIN32
Syntax: include msgbox.e
integer i = message_box(sequence msg, sequence title, object style)
Description: Display a window with the specified message and title.

The style parameter determines the combination of buttons that will be available for the user to press, plus some other characteristics. It can be an atom or a sequence.

A return value of 0 indicates a failure to set up the window.
Comments: See builtins\msgbox.e for a complete list of possible values for style and i.
Example: 
response = message_box("Do you wish to proceed?",
                       "My Application",
                       MB_YESNOCANCEL)
if response = IDCANCEL or response = IDNO then
    abort(1)
end if
