Other Libraries

The following libraries are technically not part of Phix, but are included in the standard distribution.

Sine a plain alpabetical list was starting to get out of hand, these have been reorganised into several ad-hoc subgroups:

general - maths, formatting, regex, testing, etc.
internet - internet access and file format conversion.
programming - specifically programming-related aids.
storage - data indexing and storage solutions.
user interface - GUI, graphics, and a simple beep function.
utilites (currently this just holds some dumped text)
deprecated - no plans to deliberately cripple anything, though.
