Other Libraries

The following libraries are technically not part of Phix, but are included in the standard distribution.

Since a plain alpabetical list was starting to get out of hand, these have been reorganised into several ad-hoc subgroups:

general - maths, formatting, regex, dates, testing, etc.
GUI - graphical user interfaces, and a simple beep function.
internet - internet access and file format conversion.
programming - specifically programming-related aids.
storage - data indexing and storage solutions.

Suggestions for any alternative names and/or groupings to the above five would be welcome.

deprecated - no plans to deliberately cripple anything, though.
