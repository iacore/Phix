pIUP

IUP is a multi-platform toolkit developed at Tecgraf/PUC-Rio for building graphical user interfaces. It offers high performance via native interface elements and fast learning by the user, due to the simplicity of its API.

pIUP is a simple wrapper (of the C version). The distribution includes all the required pre-built binaries: IUP, CD and IM are all needed. I include 32 and 64 bit Windows versions, but only 32 bit Linux, since at the moment I cannot test or support 64 bit Linux. A finished (precompiled) application that uses pIUP must also install the appropiate subdirectory containing the required dlls. Newer/other versions can, if required, be downloaded from:
  • CD - http://sourceforge.net/projects/canvasdraw/files/ (5.9.0.0 15/09/15)
  • IM - http://sourceforge.net/projects/imtoolkit/files/
  • IUP - http://sourceforge.net/projects/iup/files/ (3.17.0.0 30/11/15)
I went with the "dllw4" builds but feel free to try one of the others. Be advised that I found the downloads to be very flakey, often incomplete, and strongly recommend verifying with eg the 7-zip "test" button.

pIUP is (being) built from the ground up, testing and documenting each routine as it is added, hence there will be many that are not yet available. If in any doubt, please check the official documentation available at http://www.tecgraf.puc-rio.br/iup/. I have stuck with the C names, on the basis of easing both translation from C code, and searching for C and/or pIUP code snippets with just the one shot. (Besides, if you really want other names, a thin wrapper around pIUP.e should not be particularly difficult!)

Credit is due to Jeremy Cowgar and Greg Haberek; pIUP contains code cherry picked from both.

The following table will likely be broken up, possibly types and utility functions/toplevel/containers/elements/canvas draw [DEV]

Icallback - Utility function. Shorthand for call_back(routine_id(s)).
Ihandle - The interface element type.
cdCanvas - The canvas type.
IupOpen - Initialise IUP
IupImageLibOpen - Opens a library of pre-defined stock images for buttons and labels.
IupClose - Shutdown IUP
IupSetCallback - Associates a callback to an event.
IupDialog - Create a window.
IupFileDlg - Creates the File Dialog element.
IupFontDlg - Creates a Font Dialog element, for selecting a font.
IupGetColor - Shows a modal dialog which allows the user to select a color.
IupGetParam - Shows a modal dialog for capturing parameter values using several types of controls.
IupShow - Displays a dialog.
IupShowXY - Displays a dialog in a given position on the screen.
IupSetFocus - Sets the interface element that will receive the keyboard focus.
IupHide - Hides an interface element.
IupPopup - Show a dialog or menu, equivalent of a modal dialog is some toolkits.
IupSetAttribute - Sets an interface element attribute.
IupSetAttributes - Sets several attributes of an interface element.
IupGetAttribute - Retrieve an interface element attribute value.
IupSetHandle - Associates a name with an interface element.
IupGetClassName - Returns the name of the class of an interface element.
IupSetGlobal - Sets an attribute in the global environment.
IupGetGlobal - Returns an attribute value from the global environment.
IupAlarm - Show a modal dialog containing a message and up to three buttons.
IupMessage - Show a modal dialog containing a message with an OK button.
IupButton - Creates an interface element that is a button, that can contain a text and/or an image.
IupCanvas - Creates an interface element that is a canvas - a working area for your application.
IupFill - Creates void element, which dynamically occupies empty spaces always trying to expand itself.
IupFrame - Creates a native container, which draws a frame with a title around its child.
IupImage - Creates an image to be shown on a label, button, toggle, or as a cursor.
IupLabel - Creates a label interface element, which displays a separator, a text or an image.
IupList - Creates an interface element that displays a list of items.
IupLoadImage - Loads an image from a file.
IupMenu - Creates a menu element, which groups 3 types of interface elements: item, submenu and separator.
IupMenuItem - Creates an item of the menu interface element.
IupSeparator - Creates the separator interface element. It shows a line between two menu items.
IupSubmenu - Creates a menu item that, when selected, opens another menu.
IupProgressBar - Creates a progress bar control.
IupRadio - Creates a void container for grouping mutual exclusive toggles.
IupTabs - Creates a tab control (like IupZbox, but interactively visibile)
IupText - Creates an editable text field.
IupTimer - Creates a timer
IupToggle - Creates a toggle interface element
IupTree - Creates a tree containing nodes of branches or leaves.
IupHbox - Creates a void container for composing elements horizontally.
IupValuator - Creates a Valuator control (aka Scale, Slider, or Trackbar).
IupVbox - Creates a void container for composing elements vertically.
IupZbox - Creates a void container for composing elements in hidden layers with only one layer visible.
IupMap - Creates (maps) the native interface objects corresponding to the given IUP interface elements.
IupMainLoop - Start the event processing loop/user interaction
IupRecordInput - Records all mouse and keyboard input in a file for later reproduction.
IupPlayInput - Reproduces all mouse and keyboard input from a given file.
IupGetDialog - Returns the handle of the dialog that contains that interface element.
IupClipboard - Creates an element that allows access to the clipboard.
IupConfig - Initialise configuration file handler.
IupHelp - Opens the given URL.
IupDestroy - Destroys an interface element and all its children.
cdCreateCanvas - Creates a CD canvas for a virtual visualization surface (VVS).
cdCanvasLineWidth - Configures the width of the current line.
cdCanvasLineStyle - Configures the style of the current line.
cdCanvasSetForeground - Configures a new current foreground color.
cdCanvasLine - Draw a line
cdCanvasRect - Draw a rectangle with no filling
cdCanvasArc - Draw an arc
cdCanvasBegin - Starts defining a polygon to be drawn
cdCanvasVertex - Adds a vertex to the polygon definition
cdCanvasEnd - Ends the polygon's definition and draws it
cdCanvasVectorTextDirection - Define the text direction
cdCanvasMultiLineVectorText - Display multiline vector text at the specified position
cdCanvasPutImageRectRGB - Display a rectangular rgb image
wdCanvasWindow - Configures world coordinates
wdCanvasViewport - Configures a viewport in the canvas coordinate system
cdKillCanvas - Destroys a previously created canvas
