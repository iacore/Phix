power

Syntax: atom x3 = power(atom x1, atom x2)
Description: Raise x1 to the power x2
Comments: The arguments to this function must be atoms. Use sq_power() if either or both parameters might be sequences. The rules for sequence operations apply.

Powers of 2 are calculated very efficiently.
Example 1: 
? power(5, 2)
-- 25 is printed
Example 2: 
? sq_power({5, 4, 3.5}, {2, 1, -0.5})
-- {25, 4, 0.534522} is printed
Example 3: 
? sq_power(2, {1, 2, 3, 4})
-- {2, 4, 8, 16}
Example 4: 
? sq_power({1, 2, 3, 4}, 2)
-- {1, 4, 9, 16}
See Also: log, sqrt
