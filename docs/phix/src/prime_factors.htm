prime_factors

Syntax: sequence s = prime_factors(integer n)
Description: Returns a list of prime factors of n.

Note that prime_factors() always returns {} when passed a prime number.
Example: 
s = prime_factors(6) -- s is {2,3}
s = prime_factors(720) -- s is {2,3,5}
s = prime_factors(12345) -- s is {3,5,823}
