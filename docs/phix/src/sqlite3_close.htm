sqlite3_close

Definition: include pSQLite.e

integer res = sqlite3_close(sqlite3 db)
Description: Call this function with a pointer to a structure that was previously returned from sqlite3_open() and the corresponding database will by closed.

SQLITE_OK is returned if the close is successful.
If there are prepared statements that have not been finalized, then SQLITE_BUSY is returned.
SQLITE_ERROR might be returned if the argument is not a valid connection pointer returned by sqlite3_open() or if the connection pointer has been closed previously.
pwa/p2js: Not supported
Example:
Copy 
include pSQLite.e
sqlite3 db = sqlite3_open("log.sqlite")
    ...
sqlite3_close(db)
See Also: sqlite3_open
