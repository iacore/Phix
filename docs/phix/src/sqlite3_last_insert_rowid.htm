sqlite3_last_insert_rowid

Definition: include pSQLite.e

atom res = sqlite3_last_insert_rowid(sqlite3 db)
Description: Each entry in an SQLite table has a unique integer key.
The key is the value of the INTEGER PRIMARY KEY column if there is such a column, otherwise the key is generated at random.
The unique key is always available as the ROWID, OID, or _ROWID_ column.
This routine returns the integer key of the most recent insert in the database.

This function is similar to the mysql_insert_id() function from MySQL.
