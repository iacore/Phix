string

Syntax: integer i = string(object x)
Description: Return 1 if x is a string else return 0.
Comments: This serves to define the string type. You can also call it like an ordinary function to determine if an object is a string.
Example 1: 
string s
s = "hello"
Example 2: 
if string(x) then
    puts(1,x)
else
    -- x must be an atom or dword-sequence
    ? x
end if
