sum

Definition: atom res = sum(object x)
-- or --
atom res = product(object x)
Description: Add (or multiply) together all elements of x, however deeply nested.
Comments: This function may be applied to an atom or all elements of a sequence, note however that unlike most of the other math functions there are no separate sq_sum or sq_product routines.
Example: 
atom s = sum({1,2,3,4}),    -- s is 10
     p = product({1,2,3,4}) -- p is 24
Implementation: See builtins\psum.e (an autoinclude) for details of the actual implementation.
