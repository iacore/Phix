sysexec

Syntax: i1 = sysexec(string st)
Description: Similar to system_exec(), except that %COMSPEC% /C is used, so command line redirection and builtin MS_DOS commands can be used.

Note this routine has no known equivalent on RDS Eu or OpenEuphoria.

If you cannot decide between system, sysproc, system_exec, and sysexec, this table may help.
Example: 
-- executes myprog by redirecting standard input and standard output
if sysexec("p.exe \\test\\myprog.ex < indata > outdata") then
    puts(2, "failure!\n")
end if
See Also: system_exec, system
