sysproc

Syntax: sysproc(string st)
Description: Execute the specified command string without waiting for a result.
Comments: Similar to system(), except that %COMSPEC% /C is NOT used (which means no redirection, etc). Recommended for running windows GUI apps.

Note this routine has no known equivalent on RDS Eu or OpenEuphoria.

If you cannot decide between system, sysproc, system_exec, and sysexec, this table may help.
Example: 
sysproc("\"C:\\Program Files (x86)\\Phix\\pw.exe\" \"C:\\Program Files (x86)\\Phix\\pgui\"")
-- note use of double backslash in literal string to get a single backslash, and the use of
-- escaped double quotes around any path or filename containing a space
See Also: system
