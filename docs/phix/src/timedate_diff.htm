timedate_diff

Definition: include builtins\timedate.e

atom a = timedate_diff(timedate td1, timedate td2)
Description: Returns the difference in seconds between two dates. A positive value is returned when td1 is before td2.
Comments: As with timedelta, the result is intended to be used as a parameter to adjust_timedate(), and it can also be used as an argument to elapsed().
Example: 
include builtins\timedate.e
timedate birthdate = parse_date_string("1/1/1960",{"D/M/YYYY"})
printf(1,"You are %s old\n",{elapsed(timedate_diff(birthdate,date()))})
-- displays "You are 56 years, 358 days, 15 hours, 35 minutes and 22s old"
See Also: timedate, adjust_timedate, parse_date_string, elapsed
