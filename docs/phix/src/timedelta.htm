timedelta

Syntax: atom a = timedelta(atom weeks=0, days=0, hours=0, minutes=0, seconds=0, milliseconds=0, microseconds=0)
Description: returns a value in seconds and fractions of a second (idea cribbed from python)
Comments: The result of this value is intended to be used as an parameter to adjust_timedate().

milli/microseconds are probably only useful in these routines in delta*bignumber cases.
Example: 
atom fourdays = timedelta(days:=4)
timedate td = parse_date_string("1/1/2015",{"D/M/YYYY"})
td = adjust_timedate(td,fourdays)
?format_timedate(td,"Dddd, Mmmm dth, yyyy")
-- displays "Friday, January 5st, 2015"
See Also: timedate, adjust_timedate, parse_date_string, format_timedate
