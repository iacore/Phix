upper

Syntax: object x2 = upper(object x1)
Description: Convert an atom or sequence to upper case.

Should you need to, you can also invoke isupper(integer ch) to determine if a character is upper case. [ASCII only]
Example: 
s = upper("Phix")
-- s is "PHIX"
a = upper('g')
-- a is 'G'
s = upper({"Phix", "Programming"})
-- s is {"PHIX", "PROGRAMMING"}
See Also: lower
