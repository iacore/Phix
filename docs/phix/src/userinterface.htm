GUI

Libraries and components concerned with graphical user interfaces:

beep - a (somewhat painfully) simple beep procedure (console or gui).
librsvg - [draft] svg (scaleable vector graphics) handling.
pGUI - The Phix portable Graphical User Interface - 32/64 bit, Windows/Linux compatible.
xpGUI - The Phix Cross Platform Graphical User Interface - 32/64 bit, Windows/Linux/p2js/ARM compatible.

xpGUI is a planned replacement for pGUI, that will not depend on IUP and should therefore simplify installation on Linux, get rid of a couple of (Windows) niggles, and hopefully will/should also work on a Raspberry Pi (but that’s all quite some way off).
